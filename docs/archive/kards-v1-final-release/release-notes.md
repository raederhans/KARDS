# KARDS Card Forge v1.0.0

## 简体中文

这是 KARDS Card Forge 的首个稳定版。它把当前卡牌编辑器、本地工作台和发布边界固定为 `v1.0.0`。

### 主要内容

- 完整的浏览器卡牌编辑、Canvas 预览，以及 PNG、JPG、PDF 多分辨率导出。
- 外观、卡库、导出、参考四标签工作台。
- 本地卡库 CRUD、项目文件、自动草稿和结构化导出诊断。
- 参考卡搜索、筛选、稳定排序、对比、仅应用卡图、载入整卡和唯一匹配自动卡图。
- 中英文 README、应用内帮助页，以及更直接的用户提示。
- 软件版本更新为 `1.0.0`。

## English

This is the first stable KARDS Card Forge release. It fixes the current card editor, local workbench, and publication boundary at `v1.0.0`.

### Highlights

- Browser-based card editing, Canvas preview, and multi-resolution PNG, JPG, and PDF export.
- Four-tab Appearance, Library, Export, and Reference workbench.
- Local-library CRUD, project files, automatic drafts, and structured export diagnostics.
- Reference search, filtering, stable sorting, comparison, artwork-only application, full-card loading, and unique-match automatic artwork.
- English and Simplified Chinese README files, in-app help, and clearer user guidance.
- Package version advanced to `1.0.0`.
