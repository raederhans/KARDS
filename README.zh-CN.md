# KARDS Card Forge

[English](README.md) | 简体中文

在浏览器中制作、预览并导出自定义 KARDS 风格卡牌。你的卡牌内容和所选本地文件只在浏览器中处理，应用不会上传这些内容，只会从部署站点加载自身资源。

[在 Vercel 打开应用](https://kards-card-forge.vercel.app/) ·
[打开 GitHub Pages 版本](https://raederhans.github.io/KARDS/)

KARDS Card Forge 目前作为非官方、非商业同人工具维护。它不是游戏客户端、卡组工具、账号工具或官方资源下载器。软件使用范围以 [LICENSE](LICENSE) 为准。

点击顶部的 **帮助** 可查看应用内说明。帮助页会介绍首次制卡流程、保存方式、参考卡操作和本地文件行为。

## 制作第一张卡牌

1. 在左栏编辑卡牌字段。
2. 上传卡图；或者打开 **参考**，选择一张卡牌，再点击 **仅应用卡图**。
3. 拖动卡图调整位置，使用鼠标滚轮缩放。
4. 在 **外观** 中调整纸张纹理，或载入本地风格包。
5. 打开 **导出**，确认检查结果，然后导出 PNG、JPG 或 PDF。

预览使用固定的 `500 × 702` 布局。选择 `2×` 或 `3×` 导出时，应用会按目标分辨率重新绘制，而不是简单放大预览图。

## 工作台

右侧工作台包含四个标签：

| 标签 | 用途 |
| --- | --- |
| **外观** | 调整纸张纹理，或载入本地风格包。 |
| **卡库** | 下载或打开项目文件，并管理本地卡库。 |
| **导出** | 选择格式、分辨率、曝光、对比度和保存位置。 |
| **参考** | 搜索、筛选、对比和使用随应用提供的参考卡。 |

在 **参考** 中，选择参考卡只会切换对比卡。**仅应用卡图** 只替换卡图，**载入整张卡牌（覆盖当前）** 会替换当前整张卡牌。自动卡图只在找到唯一匹配时生效，绝不会覆盖你上传或主动选择的卡图。

## 保存工作

三种保存方式用途不同：

- 自动草稿只在当前浏览器、当前站点保存轻量卡牌数据，不保存嵌入卡图，包括上传或从参考卡应用的卡图。Vercel 和 GitHub Pages 的站点存储不互通。
- 项目文件（`.card.json`）保存完整的可编辑卡牌和嵌入卡图，适合稍后继续编辑同一张卡牌。
- 本地卡库（`card-forge-library.json`）把可复用卡牌保存在你选择的文件夹中，支持新增、载入、更新和删除，但不保存嵌入卡图。

## 浏览器支持

文件夹访问依赖浏览器的 File System Access API。如果浏览器不支持，导出会改用普通浏览器下载，本地卡库文件夹则无法打开。

如果浏览器支持文件夹访问，但不支持 Web Locks，本地卡库会进入只读状态：可以浏览和载入，但不能新增、更新或删除。记住文件夹不等于永久获得权限；打开或写入时，浏览器仍可能再次询问。

## 本地风格包

本地风格包是一个包含 `kards-asset-pack.json` 及其所列图片和字体的文件夹。在 **外观** 中选择整个文件夹。应用只会在当前浏览器会话中使用这些文件，不会上传这些文件。

可从[清单示例](docs/reference/asset-pack-manifest.example.json)开始。请只使用你有权使用的文件。

## 本地开发

项目使用 React、TypeScript、Vite 和 Canvas 2D。安装依赖并启动本地服务器：

```bash
npm ci
npm run dev
```

CI 基线为 Node.js 22 和 Python 3.12。运行完整仓库检查前，请先安装 Python 测试依赖：

```bash
py -3 -m pip install -r requirements-dev.txt  # Windows
python3 -m pip install -r requirements-dev.txt # macOS 或 Linux
npm run validate
```

常用命令：

| 命令 | 用途 |
| --- | --- |
| `npm test` | 运行一次 Vitest 测试。 |
| `npm run typecheck` | 检查 TypeScript 项目。 |
| `npm run build` | 构建并检查最终公开的 `dist` 产物。 |
| `npm run validate` | 运行测试、私有工具契约、类型检查、构建和产物边界检查。 |

## 项目结构

```text

  App.tsx                  编辑器状态和参考卡连接
  components/              字段、预览、工作台和卡库界面
  canvas/                  固定卡面几何和渲染器
  assetPack.ts             内置与本地风格包载入
  exportCard.ts            PNG、JPG 和 PDF 导出管线
  localLibrary.ts          本地卡库文件操作
  storage.ts               轻量浏览器草稿
tools/                     构建边界和校准检查
docs/active/               当前路线图和 worktree 注册表
docs/archive/              已完成的实施记录
```
